# Odin-Rock-Paper-Scissors Game

This is a simple JavaScript console-based Rock Paper Scissors game. The game runs in the browser console and allows a user to play five rounds against the computer.

# How to Play

Open the browser and navigate to the developer console (Press F12 or Ctrl + Shift + I and go to the Console tab).

Run the JavaScript code provided in rock_paper_scissors.js.

The game will prompt you to enter rock, paper, or scissors.

The computer randomly selects its choice.

The game determines the winner for each round and keeps track of the score.

After five rounds, the final winner is announced based on the total score.

# Features

Fully interactive console-based game.

Validates user input to ensure only valid choices are accepted.

Tracks the score and declares an overall winner after five rounds.

Uses JavaScript functions for modularity and clarity.

# Requirements

A modern web browser with JavaScript enabled.

Basic understanding of how to open the browser console.
